# Sertifikalar — Belge Görselleri ve PDF'ler

Bu klasöre kalite/üyelik belgelerinizin görsel (JPG/PNG) ve PDF kopyalarını koyun.

## Beklenen dosya adları:

### Görsel önizlemeler (kart arka planında):
- `iso-9001.jpg` — ISO 9001:2015 belgesi (önizleme)
- `ce.jpg` — CE belgesi
- `tsek.jpg` — TSEK onay belgesi
- `tbdy-uyum.jpg` — TBDY 2018 uygunluk beyanı (varsa)

### İndirilebilir PDF'ler (tam belge):
- `iso-9001.pdf`
- `ce.pdf`
- `tsek.pdf`

## Format önerisi:
- **JPG** 72dpi, ~800px genişlik (web önizleme için)
- **PDF** orjinal belge (indirme için)

## Not:
`/kurumsal/belgelerimiz/` sayfası hazırlanırken bu klasördeki dosyalar otomatik kullanılacak. Boş bırakılırsa generic SVG rozet görünür.
