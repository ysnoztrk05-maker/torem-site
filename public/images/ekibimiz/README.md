# Ekibimiz — Personel Fotoğrafları

Bu klasöre ekip üyesi fotoğraflarını koyun (kurumsal portre).

## Önerilen dosya isimlendirme:
`ad-soyad.jpg` formatında, küçük harf ve tire ile:
- `yasin-ozturk.jpg`
- `ahmet-kaya.jpg`
- `ayse-demir.jpg`
- `...`

## Format önerisi:
- **JPG** 1:1 kare veya 4:5 dikey (portre)
- Boyut: ~600x600px (kare) veya ~600x750px (dikey)
- Kurumsal: nötr zemin, iş kıyafeti, profesyonel aydınlatma
- Net, kesintisiz yüz odaklı çerçeveleme

## Her üye için istenecek bilgiler (ekibimiz sayfasına):
- Ad Soyad
- Unvan (ör. "Kurucu & Saha Müdürü", "Kıdemli Jeoloji Mühendisi")
- Mezuniyet (üniversite / bölüm)
- Deneyim yılı
- Uzmanlık alanı (ör. "Fore Kazık ve Derin Temel Uygulamaları")
- Kısa tanıtım (1-2 cümle)
- İsteğe bağlı: LinkedIn URL

Bu bilgileri liste halinde bana verdiğinde sayfayı oluştururum.

## Not:
Yüz görseli olmayan üyeler için default silüet SVG kullanılır.
