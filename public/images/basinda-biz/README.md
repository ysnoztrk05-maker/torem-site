# Basında Biz — Medya/Röportaj Görselleri

Bu klasör `/kurumsal/basinda-biz/` sayfası için (gelecek).

## Gelecekte buraya girecek:
- Gazete/dergi makale ekran görüntüleri
- Fuar/etkinlik katılım fotoğrafları
- Röportaj kapak görselleri
- Medya logoları

## Format önerisi:
- Dosya: `yyyy-mm-yayin-adi.jpg` formatında (ör. `2025-03-insaat-yatirim.jpg`)
- Boyut: ~1200px genişlik max

## Not:
Basında Biz sayfası şimdilik oluşturulmadı. Medya materyaliniz varsa bildirin, sayfa açılır.
